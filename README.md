# Velvet Match — MERN Project Base

Base inicial de un proyecto MERN diseñada siguiendo buenas prácticas de organización, documentación y control de versiones para evolucionar hacia una aplicación full-stack lista para producción.

## Descripción del proyecto
Este repositorio representa la base inicial del proyecto Velvet Match, una plataforma web tipo SaaS orientada a negocios del área de belleza y estética personal.  
El objetivo es establecer una estructura profesional desde el inicio, permitiendo que el proyecto evolucione durante el semestre hasta convertirse en un MVP funcional listo para producción.

El sistema busca digitalizar la gestión de negocios mediante la creación automatizada de sitios web, administración de citas, roles de usuario y herramientas de gestión interna. 

---

## Objetivo general
Desarrollar una plataforma web multi-tenant que permita a negocios de belleza gestionar sus servicios, empleados y citas dentro de un entorno centralizado, utilizando el stack MERN como base tecnológica.

---

## Tecnologías principales
- MongoDB
- Express.js

- Node.js
